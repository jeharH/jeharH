## Projetos que estou construindo

### Analytics para credito e risco

Analises voltadas a indicadores de carteira, inadimplencia, perda esperada, comportamento de clientes e leitura de risco.

**Stack prevista:** Python, SQL, Pandas, visualizacao de dados

### Pipeline de dados com PySpark

Projeto para ingestao, tratamento e transformacao de dados em ambiente distribuido, com foco em organizacao e reprodutibilidade.

**Stack prevista:** PySpark, Databricks, SQL

### Dashboard de indicadores

Visualizacao de metricas de negocio com leitura executiva, filtros e storytelling orientado a decisao.

**Stack prevista:** Power BI ou Looker Studio, SQL, modelagem de dados
